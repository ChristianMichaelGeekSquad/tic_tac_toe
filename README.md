# Tic Tac Toe

This game was built in 2 days, so please pardon any inconsistencies I might have missed out in the during the coding process. 

I have added a **minimax** algorithm to this project. I took code inspiration from ChatGPT and baked it into this mini game. At the momen, it is impossible to beat computer but I can modify it if needed to accommodate for different diffuclty levels so it's more interesting


## Steps to build this project
* Ensure you have you environments all set up
    * node version ``` >= 20 ``` should suffice
    * Run `yarn` or `yarn install` to install all dependencies required to build the project successfully
    * Run `npx pod-install` - can run fro root with cd'n into the `ios` dir  or `cd ios && pod install`
    * Open the `.xcworspace` file for this project on xcode and build it on your device
